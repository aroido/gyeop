# GYEOP Question Pack Design

Read `docs/product/question-pack-spec.md` and the current loop in `docs/product/core-feature-priority.md`.

## Design a pack

1. Set the relationship, tone, sensitivity, and public or 1:1 share default.
2. Write exactly 10 active A/B cards with one Signature card.
3. Keep choices balanced, concrete, non-leading, and natural for both owner and named-subject visitor wording.
4. Add relation tags and a short comparison insight without fixed personality labels.
5. Preview wording within mobile line limits.

## Visitor sampling

1. Assign the Signature card.
2. Exclude cards this visitor already answered.
3. Choose two distinct cards from the lowest-response remaining group, expanding only when necessary.
4. Show comparison after the required three cards; offer two additional under-sampled cards only as a secondary CTA.

Never reveal owner choices before submission. Update `docs/product/question-pack-spec.md` only when a reusable rule changes.
